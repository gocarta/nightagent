# nightagent
> Automatically Detect and Repair Common Firewall Issues

## install
```
pip install nightagent
```

## usage (in cli)
```sh
> nightagent --device-type paloalto_panos --hostname 192.168.1.1 --username admin monitor
> password: 
> secret: 
```

## usage (in uvx)
If you have [uv](https://docs.astral.sh/uv/) installed, you can skip the install step and go straight to usage with:
```sh
uvx nightagent monitor
```
